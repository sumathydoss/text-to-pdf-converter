## How to Use

1. **Upload a File** - Drag & drop a TXT or HTML file, or click to browse
2. **Or Paste Content** - Paste text or HTML directly in the textarea
3. **Add Title (Optional)** - Enter a PDF title
4. **Convert** - Click "Convert to PDF" to download your file
5. **Theme Toggle** - Click the moon/sun icon to switch themes

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **UI Framework**: Bootstrap 5
- **PDF Generation**: jsPDF + html2canvas
- **Icons**: Bootstrap Icons
